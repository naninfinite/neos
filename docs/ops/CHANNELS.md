# AgentChatTR Channels

Recommended channel structure for NEOS.

## #general

Top-level orchestration.
Use for kickoff, role reminders, and task assignment.

## #architecture

Claude-led spec discussion and architecture checks.

## #implementation

Codex task execution and build-focused discussion.

## #validation

Gemini review, contradiction checks, QA, and acceptance checks.

Visual documentation decisions, screenshot selection, and PDF export updates.

Short locked decisions only.
